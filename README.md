# 🍽️ Zomato Sales Analysis Dashboard

## 📌 Project Overview
The **Zomato Sales Analysis Dashboard** is a Power BI project developed to analyze restaurant sales performance, customer behavior, ratings, and order trends.

This dashboard provides business insights that help understand:
- Sales performance
- Customer preferences
- Restaurant ratings
- Order trends
- Revenue generation
- City-wise analysis

The project uses interactive visualizations and data analytics techniques to transform raw restaurant data into meaningful business insights.

---

# 🎯 Business Problem
Food delivery platforms generate massive amounts of transactional and customer data daily.

Businesses often struggle to:
- Identify top-performing restaurants
- Understand customer ordering patterns
- Track sales trends
- Improve customer satisfaction
- Analyze city-wise performance

This dashboard helps stakeholders make data-driven decisions using interactive analytics.
